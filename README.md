Project Overview

This project focuses on creating realistic lighting and cinematic post-processing effects in Unreal Engine. The goal is to enhance visual quality, mood, and atmosphere in a game environment.

Lighting Setup

The lighting system was designed to add depth, realism, and balance within the scene.

Techniques Used
Directional Light

Adjusted intensity and angle to simulate natural sunlight

Enabled dynamic shadows for realistic lighting

Light Color & Temperature

Tuned color temperature to match the scene mood

Shadow Settings

Enabled soft shadows to avoid harsh lighting

Ambient Lighting

Balanced indirect lighting to reduce dark areas

Post-Processing Effects

Used a Post-Processing Volume to control the overall visual tone.

Effects Applied

Bloom

Adds glow to bright areas for a cinematic look

Depth of Field

Enhances focus on primary objects

Ambient Occlusion

Improves depth perception with soft shadowing

Color Grading

Adjusted contrast, saturation, and color balance

Scene Testing

Tested the scene in Play Mode

Verified lighting and post-processing integration

Ensured visual effects enhance mood without affecting gameplay

Conclusion
